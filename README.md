# tax-calc

Small utility for yearly taxes calculation

## Setup

Requirements:

- node & npm

Install dependencies with:

```bash
npm install
```

## Development

Run an application with:

```bash
npm run dev
```

## Built with

- [react-hook-form](https://react-hook-form.com/)
- [zod](https://zod.dev/)
- [zustand](https://zustand.docs.pmnd.rs/)
